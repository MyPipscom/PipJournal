## Tech Stack

- **Frontend**: Vanilla HTML, CSS, JavaScript
- **Storage**: localStorage (browser-based)
- **Hosting**: Netlify
- **No backend required** - runs entirely in the browser

## Getting Started

### Option 1: Use Online
Visit [pipjournal.netlify.app](https://pipjournal.netlify.app) (after deployment)

### Option 2: Run Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/YOUR_USERNAME/pipjournal.git
   ```
2. Open `index.html` in your browser

### Option 3: Deploy Your Own
1. Fork this repository
2. Connect to Netlify
3. Deploy automatically

## Usage

1. **Add Trades**: Click "+ New Trade" to log your trades
2. **Track Progress**: View your equity curve and stats on the Dashboard
3. **Analyze**: Use Analytics to find patterns in your trading
4. **Journal**: Write daily reflections in the Journal section
5. **Build Playbook**: Document your strategies with entry rules

## Data Storage

All data is stored locally in your browser using localStorage. To backup your data:
1. Go to Settings
2. Click "Export Full Backup"
3. Save the JSON file

To restore:
1. Go to Settings
2. Click "Import Backup"
3. Select your backup file
